## 📝 Descripción
Esta es una aplicación interactiva diseñada para fanáticos de los cómics que permite buscar y explorar información detallada sobre héroes y villanos. Utilizando la **SuperHero API**, la aplicación recupera datos biográficos, estadísticas de poder y afiliaciones de personajes de DC y Marvel.

Este proyecto destaca por la implementación de **pruebas unitarias**, asegurando que los componentes y la lógica de filtrado funcionen correctamente bajo distintos escenarios.

## 🛠️ Tecnologías Utilizadas
- **React.js**: Construcción de la interfaz y gestión de componentes.
- **Jest**: Framework de testing para asegurar la integridad del código.
- **Axios**: Comunicación asíncrona con la API.
- **GitHub Pages**: Despliegue automático de la aplicación.

## 🧪 Calidad y Testing
Para garantizar la estabilidad del proyecto, se incluyeron pruebas con **Jest** enfocadas en:
- **Renderizado de Componentes:** Verificación de que las tarjetas de héroes se carguen correctamente.
- **Lógica de Búsqueda:** Pruebas para asegurar que los filtros devuelvan los resultados esperados según el input del usuario.
- **Manejo de Errores:** Validación de la interfaz cuando no se encuentran resultados en la API.

## 📖 Ejemplos de Uso
**Buscador Dinámico**
- Escribe el nombre de un personaje (ej. "Batman"). La aplicación consulta la API y muestra todas las versiones disponibles del héroe de forma inmediata.

**Análisis de Poder**
- Consulta las Powerstats para comparar niveles de Inteligencia, Combate y Poder entre diferentes personajes, visualizando los datos de forma estructurada.

**Ficha Técnica**
- Acceso a datos detallados como nombre real, editora (Marvel/DC) y características físicas.
